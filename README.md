# Conversor de Unidades (Decimal, Binário, Octal e Hexadecimal)
## Descrição

Um conversor de unidades numéricas que permite conversão entre os sistemas de numeração Decimal, Binário, Octal e Hexadecimal. Ferramenta útil para estudos de programação e desenvolvimento de software.

## Funcionalidades

- ✅ Conversão de Decimal para Binário, Octal e Hexadecimal
- ✅ Conversão de Binário para Decimal, Octal e Hexadecimal
- ✅ Conversão de Octal para Decimal, Binário e Hexadecimal
- ✅ Conversão de Hexadecimal para Decimal, Binário e Octal


## Tecnologias

- [Python]

## Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/Danielius7er/conversordeNum.git
```

2. Navegue até o diretório e o ficheiro src:
```bash
cd conversordeNum
cd src
```
3. Rodar o código
```bash
    python main.py
    python teste_main.py
```
3. [Instruções específicas de execução]

## Exemplos

- Decimal 10 → Binário: 1010
- Decimal 10 → Octal: 12
- Decimal 10 → Hexadecimal: A
